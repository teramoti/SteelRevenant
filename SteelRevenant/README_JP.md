# SteelRevenant

DirectX 11 / DirectXTK を使用した3Dアクションゲームです。

## 作品概要

近接戦闘をベースとしたウェーブサバイバル形式の3Dアクションゲームです。
制限時間内に押し寄せる敵を倒し続けることがゲームの目的です。
3種類のステージが用意されており、ステージごとに敵の構成と制限時間が異なります。

## 操作方法

| キー / マウス | 操作 |
|---|---|
| WASD | 移動 |
| マウス移動 | カメラ視点操作 |
| 左クリック | 攻撃 |
| 右クリック | ガード |
| Shift（長押し） | スライド回避 |
| Tab | ロックオン切替 |
| Esc | ポーズ / 前の画面に戻る |

## ゲームの目的

各ステージの制限時間が終了するまで戦闘を継続してください。
時間内に敵を多く倒すほどスコアが高くなります。

## ビルド・実行方法

1. Visual Studio 2019 または 2022 で `SteelRevenant.sln` を開きます。
2. 構成を `Debug | x64` または `Release | x64` に設定してビルドします。
3. 音声ファイルは `Assets/Audio/` 以下に配置されています。音声が再生されない場合は、このフォルダの存在を確認してください。
4. スプライトフォントファイル（`.spritefont`）が `Source/` フォルダと同じ階層に存在することを確認してください。存在しない場合はフォールバック処理でデフォルトフォントが読み込まれます。

## 実装要素

- 近接攻撃・コンボ・ガード・スライド回避
- ロックオンシステム
- 複数種類の敵AI（近接・レーザー射撃・重装甲）
- A\*経路探索による敵の移動
- ウェーブサバイバル形式のステージ進行（全3ステージ）
- スコア集計とリザルト画面
- ミニマップ表示
- ポーズメニュー
- BGM / SE 再生（DirectXTK Audio 使用）
- KH風3D追従カメラ（XZ/Y分離補間・ロックオン追従・壁遮蔽回避）
