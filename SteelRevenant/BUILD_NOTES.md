# ビルドに関する注意事項

## 文字コード

ソースファイルはすべて UTF-8 で保存してください。
Shift_JIS で保存されたファイルが混在すると C4819 警告やコンパイルエラーの原因になります。

## スプライトフォント

`.spritefont` ファイルが実行ファイルと同じ階層、または `Source/` フォルダに存在することを確認してください。
ファイルが見つからない場合はフォールバック処理が実行されますが、一部の文字が表示されないことがあります。

## 音声ファイル

音声ファイルは `Assets/Audio/` 以下に配置してください。
AudioSystem は実行時に `Assets/Audio` を起点として SE / BGM ファイルをロードします。
ファイルが存在しない場合は無音で動作します。

## 動作確認済み構成

- Windows 11
- Visual Studio 2022
- 構成: Debug / Release、プラットフォーム: x64
